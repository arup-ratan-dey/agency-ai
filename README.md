# 🌐 Digital Agency Landing Page (React + Tailwind)

## 📌 Overview

This project is a modern **Digital Agency Landing Page** built using **React**, **Tailwind CSS**, and **Framer Motion** for animations.
It presents a complete business website layout including hero section, services, portfolio, team members, contact form, and footer.

The goal of this project was to create a responsive, visually appealing, and interactive frontend website with smooth animations and dark/light theme support.

---

## ✨ Features

* 🌗 Dark / Light theme toggle with local storage support
* 🎞️ Smooth animations using Framer Motion
* 🧭 Responsive navbar with mobile sidebar
* 🎯 Hero section with gradient text and visuals
* 🏢 Trusted companies logo section
* 🛠️ Services cards with hover glow effect
* 📂 Portfolio / Work showcase grid
* 👥 Team members section
* 📩 Functional contact form using Web3Forms API
* 🔔 Toast notifications for form submission
* 🖱️ Custom animated cursor (dot + ring effect)
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* Framer Motion
* Web3Forms API
* React Hot Toast
